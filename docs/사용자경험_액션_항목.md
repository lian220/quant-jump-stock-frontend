# 프론트엔드 UX 개선 실무 가이드 (Summary)

> 2026-02-18 감사 결과 기반. 상위 태스크는 루트 `docs/할일.md`에서 관리.

## 1. 긴급 수정 (Critical)
*   **온보딩/대시보드 500 에러**: 로그인 후 `/onboarding` 진입 시 발생하는 크래시 해결. 클라이언트 사이드 데이터 페칭 또는 SSR 로직 점검.

## 2. UI/UX 세부 항목
| 페이지 | 항목 | 세부 내용 |
| :--- | :--- | :--- |
| **공통** | 수익률 시각화 | 단순 텍스트(%)를 미니 차트나 그래프가 포함된 썸네일로 교체하여 신뢰도 향상 |
| **공통** | BETA 뱃지 | 헤더 뱃지 크기 10% 축소, 페이지 내부의 대형 'BETA 경고' 배너 제거 |
| **전략 마켓** | 기본 정렬 | 마이너스 수익률 전략이 상단에 오지 않도록 수익률/인기순 기본값 설정 |
| **네비게이션** | BottomNav 라벨 | "분석" -> "AI 추천" 등 더 직관적인 용어로 변경 |

## 3. 마케팅/전략
*   **소셜 프루프**: 실제 투자 성공 사례나 사용자 후기를 위한 카드 레이아웃 추가.
*   **브랜드 일관성**: 메타 태그(`title`, `description`) 및 OG 이미지를 "AI가 찾고, 전략이 검증합니다" 테마로 통일.
