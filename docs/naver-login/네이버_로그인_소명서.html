2. 서비스 성격 명확화

본 서비스는 「자본시장과 금융투자업에 관한 법률」상 금융투자업(투자자문업, 투자일임업)에 해당하지 않으며, 다음과 같은 정보 제공 서비스입니다.

✅ 본 서비스가 제공하는 것

  • AI 알고리즘 기반 주식 데이터 분석 결과 제공
  • 과거 데이터 기반 백테스트 통계 정보 제공
  • 기술적 지표 및 퀀트 분석 데이터 제공
  • 투자 참고 자료로서의 정보 제공

❌ 본 서비스가 제공하지 않는 것

  • 특정 종목의 매수/매도 권유 또는 투자 자문
  • 투자 판단 대행 또는 투자 일임
  • 자동 매매 기능 (증권사 계좌 연동 없음)
  • 개별 맞춤형 투자 상담 또는 포트폴리오 관리

5. 기술적 제한사항

본 서비스는 다음과 같은 기술적 제한을 두어 금융투자업과의 혼동을 방지합니다:

  • ❌ 증권사 API 연동 없음 (계좌 조회/매매 불가)
  • ❌ 자동 매매 기능 없음
  • ❌ 실시간 매매 신호 알림 없음
  • ❌ 개인 맞춤형 포트폴리오 관리 없음

6. 결론

본 서비스는 데이터 분석 결과를 투자 참고 자료로 제공하는 정보 서비스이며, 투자 자문이나 일임을 하는 금융투자업에 해당하지 않습니다.

관련 법률을 준수하기 위해 이용약관, 면책 조항, 용어 개선 등 필요한 모든 조치를 완료하였으며, 네이버 로그인 연동 승인을 요청드립니다.
