# 랜딩 CTA 우선순위 및 신뢰요소 적용안

## 적용 목표
- 1순위 CTA를 `무료 회원가입`으로 통일해 가입 행동을 명확히 한다.
- 가입 전 신뢰 형성을 위해 사회적 증거와 운영 지표를 노출한다.

## 적용 내용
- 히어로 Primary CTA: `무료 회원가입` (`/signup`)
- 히어로 Secondary CTA: `AI 분석 종목 보기` (`/recommendations`)
- 하단 CTA Primary도 동일하게 `무료 회원가입`으로 통일

## 신뢰요소(사회적 증거) 보강
- 최근 분석 건수(가능 시 실데이터) 노출
- 업데이트 주기 안내(영업일 기준 자동 업데이트)
- 가입 전 체험 링크(샘플 분석 리포트 보기) 제공

## 기대 효과
- 사용자가 첫 화면에서 다음 행동을 즉시 이해
- 실제 운영 신호를 통해 초기 신뢰 장벽 완화
- 가입 전 가치 체감(샘플 확인) → 가입 전환 흐름 강화
