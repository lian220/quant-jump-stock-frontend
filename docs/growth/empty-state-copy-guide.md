# Empty / Placeholder 카피 가이드

## 목적
- `-`, `N/A` 같은 플레이스홀더 노출을 줄이고 행동 유도형 안내로 통일한다.
- 모든 공개 화면에서 동일한 톤(상황 설명 + 다음 행동)을 유지한다.

## 공통 원칙
- 숫자 미집계 상태: `집계중`, `데이터 준비중`
- 데이터 없음 상태: 원인 + 대체 액션을 함께 제공
- 에러 상태: 문제 안내 + 재시도 버튼 + 보조 액션(필터 초기화/다른 화면 이동)

## 화면별 기준 카피
- `전략`: "조건에 맞는 전략이 없습니다" + 필터 조정 유도
- `뉴스`: "뉴스가 아직 준비되지 않았습니다" 또는 검색 조건 안내 + 최신 뉴스 보기
- `종목`: "조건에 맞는 종목이 없습니다" + 추천 검색 키워드 예시 + 분석 화면 이동
- `AI 분석`: "분석 데이터가 없습니다" + 최신 데이터 보기 + 종목 탐색 이동

## 구현 지침
- 공통 컴포넌트 `StateMessageCard`를 사용해 제목/설명/CTA 패턴 통일
- 지표 카드 값이 비어 있으면 `-` 대신 `데이터 준비중` 사용
- 홈 미니 대시보드 값이 비어 있으면 `집계중` 표시
