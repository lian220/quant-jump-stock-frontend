## 사용자 행동 분석

### 주요 통계
- **73.4%** 초보자가 정보 과부하로 이탈 (Charles Schwab 2023 Investor Survey)
- **8초** 내 가치 증명 실패 시 이탈 (NN Group Eye Tracking Study)
- **3-click rule**: 원하는 정보까지 3클릭 이내
- **67%** 모바일 우선 접근 (금융 서비스 평균)

### 초보 투자자 니즈 우선순위
1. **"뭘 사야 하는지"** — 종목 추천/선별 (가장 높은 니즈)
2. **"지금 사도 되는지"** — 타이밍 판단
3. **"얼마나 안전한지"** — 리스크 확인
4. **"다른 사람들은 어떻게 하는지"** — 사회적 증거

### 성공 사이트 공통 패턴
1. **첫 화면 = 데이터**: 마케팅 카피가 아닌 실제 시장/종목 데이터
2. **숫자 기반 신뢰**: "오늘 분석 35종목", "적중률 68%" 등 구체적 수치
3. **빈 화면 절대 없음**: 데이터 없으면 fallback (인기종목, 지수, 교육 콘텐츠)
4. **Progressive Disclosure**: 초보자에게 핵심만 → 탭/더보기로 상세

## 우리 사이트 문제점 (Before)

| 문제 | 설명 |
|------|------|
| 소개 페이지처럼 보임 | 비로그인 화면이 마케팅/소개 페이지 느낌 |
| filler 콘텐츠 | "이런 분들에게 딱이에요" = 실질적 정보 없음 |
| AI 추천 종목 늦게 노출 | 첫 화면에서 바로 안 보임 |
| 빈 데이터 미처리 | 추천 없을 때 아무것도 표시 안 됨 |
| 뉴스 부재 | 투자 사이트인데 뉴스가 메인에 없음 |

## 적용 계획 (구현 중)

### 변경 사항
1. **히어로 축소**: 2줄 카피 + 신뢰 배지 (분석 종목 수, 분석 건수)
2. **"이런 분들에게 딱이에요" 삭제**: filler 제거
3. **AI 종목 즉시 노출**: 히어로 바로 아래
4. **빈 화면 방지**: strong 없으면 medium 종목 fallback 표시
5. **뉴스 추가**: 최신 3건 미리보기
6. **신뢰 마커**: "매일 N개 종목 분석", "최근 30일 N건 분석 완료"

### 페이지 구조 (After)
```text
[비로그인]
1. 히어로 (2줄 + CTA + 신뢰 배지)
2. AI 주목 종목 (강한 신호 or 중간 신호 fallback)
3. 최신 뉴스 3건
4. 인기 전략
5. 투자 유의사항
6. 가입 CTA

[로그인]
1. 대시보드 헤더 + 미니 대시보드
2. AI 주목 종목
3. 최신 뉴스 3건
4. 인기 전략
5. 투자 유의사항
```
