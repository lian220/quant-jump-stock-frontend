## 1. 현황 요약

### 총평: B- (기본기 확보, 성장 기반 구축 중)

> 1차 리뷰(C+) → P0/P1 개선 → 2차 홈 재구조화 완료 후 등급 상향

**강점**

- AI 분석 + 전략 마켓플레이스 + 백테스트라는 3축 차별화 명확
- 홈페이지 정보 위계: 핵심 차별점이 최상단에 배치 (2차 개선)
- 히어로 메시지 "AI가 찾고, 전략이 검증합니다" — 경쟁사와 구분되는 포지셔닝
- 다크 테마 기반 핀테크 톤앤매너 일관성
- PWA 지원, 네이버 로그인 등 기술적 기반 양호
- 백테스트 소프트게이트 전환 유도 전략 우수
- 실시간 API 데이터 연동 (분석 종목, 등급 분포, 평균 점수)

**잔존 약점**

- 소셜 프루프(사용자 후기, 성공 사례) 여전히 전무 → P2-1
- 종목 상세 페이지에 주가 차트 미구현 → P2
- 온보딩 플로우 부재 → P2-2
- 브랜드 태그라인 전체 터치포인트 미통일 → P2-3

## 3. 경쟁 서비스 대비 포지셔닝

| 비교 항목 | Alpha Foundry | 토스증권 | 카카오페이증권 | 증권플러스 |
|-----------|:---:|:---:|:---:|:---:|
| 첫 화면 가치 전달 | ~~텍스트만~~ → 실시간 데이터 + 차별점 | 즉시 시세 | 즉시 시세 | 즉시 시세 |
| 전략 마켓플레이스 | **있음 (핵심 차별점)** | 없음 | 없음 | 없음 |
| AI/퀀트 분석 | **있음** | 제한적 | 없음 | 없음 |
| 백테스트 | **있음** | 없음 | 없음 | 없음 |
| 주가 차트 | 부분 (종목 상세 미구현) | 있음 | 있음 | 있음 |

## 4. 개선 체크리스트

### P0: 즉시 (사용자 이탈 방지) - 예상 1-3일

- [x] **P0-1. 모바일 홈 히어로 축소** ✅
  - 현재: 뱃지 + 히어로 텍스트 + 설명 3줄 + CTA 2개 (전체 첫 화면 차지)
  - 목표: 히어로를 50% 이하로 축소, AI 주목 종목 TOP 3을 첫 화면에 노출
  - 파일: `src/app/page.tsx`

- [x] **P0-2. 홈 마이너스 수익률 전략 비노출** ✅
  - 현재: 백테스트 성과 섹션에 -2.64%, -7.86% 등 표시
  - 목표: 플러스 수익률 전략만 홈에 노출하거나, 해당 섹션 제거
  - 파일: `src/app/page.tsx`

- [x] **P0-3. Recommendations 모바일 필터 접기** ✅
  - 현재: 날짜 피커 + 정렬 + 카운트 + Beta 배너 + 시간 안내가 모두 펼쳐진 상태
  - 목표: 필터를 접히는(collapsible) 형태로, 종목 카드 즉시 노출
  - 파일: `src/app/recommendations/page.tsx`

- [x] **P0-4. Beta 경고 배너 톤다운** ✅
  - 현재: 노란색 큰 배너 "점수가 낮게 표시됩니다"
  - 목표: 작은 인라인 뱃지 or 툴팁으로 축소. "기술적 지표 기반 분석 (AI 통합 예정)" 등 중립적 표현
  - 파일: `src/app/recommendations/page.tsx`

## 5. 핵심 원칙 (개선 작업 시 참고)

1. **3초 룰**: 사용자는 3초 안에 "이 서비스가 나에게 가치 있는가"를 판단함
2. **모바일 퍼스트**: 한국 사용자 80%+ 모바일. 모바일 경험이 곧 서비스 경험
3. **부정 > 긍정**: 마이너스 수익률 1개가 플러스 10개보다 인상에 강하게 남음 (손실 회피 편향)
4. **빈 공간 = 미완성**: N/A, 빈 카드, 데이터 없음은 "아직 준비 안 된 서비스" 인상
