## 1. 현황 요약

### 총평: C+ (잠재력 있으나 구조적 문제 존재)

**강점**

- AI 분석 + 백테스트라는 명확한 차별화 포인트
- 다크 테마 기반 핀테크 톤앤매너 일관성
- PWA 지원, 네이버 로그인 등 기술적 기반 양호
- 백테스트 소프트게이트 전환 유도 전략 우수

**약점**

- 모바일 첫 화면에 실질 콘텐츠 부재 (히어로 텍스트만 노출)
- 마이너스 수익률 전략이 메인 페이지에 노출
- 소셜 프루프(사용자 후기, 성공 사례) 전무
- 종목 상세 페이지에 가격/차트 등 기본 정보 부재
- BETA 메시징이 과도하여 미완성 인상

## 4. 개선 체크리스트

### P0: 즉시 (사용자 이탈 방지) - 예상 1-3일

- [ ] **P0-1. 모바일 홈 히어로 축소**
  - 현재: 뱃지 + 히어로 텍스트 + 설명 3줄 + CTA 2개 (전체 첫 화면 차지)
  - 목표: 히어로를 50% 이하로 축소, AI 주목 종목 TOP 3을 첫 화면에 노출
  - 파일: `src/app/page.tsx`

- [ ] **P0-2. 홈 마이너스 수익률 전략 비노출**
  - 현재: 백테스트 성과 섹션에 -2.64%, -7.86% 등 표시
  - 목표: 플러스 수익률 전략만 홈에 노출하거나, 해당 섹션 제거
  - 파일: `src/app/page.tsx`

- [ ] **P0-3. Recommendations 모바일 필터 접기**
  - 현재: 날짜 피커 + 정렬 + 카운트 + Beta 배너 + 시간 안내가 모두 펼쳐진 상태
  - 목표: 필터를 접히는(collapsible) 형태로, 종목 카드 즉시 노출
  - 파일: `src/app/recommendations/page.tsx`

- [ ] **P0-4. Beta 경고 배너 톤다운**
  - 현재: 노란색 큰 배너 "점수가 낮게 표시됩니다"
  - 목표: 작은 인라인 뱃지 or 툴팁으로 축소. "기술적 지표 기반 분석 (AI 통합 예정)" 등 중립적 표현
  - 파일: `src/app/recommendations/page.tsx`

  - 목표: 현재가, 전일 대비 등락률, 시가총액 등 기본 시세 정보 추가
  - 파일: `src/app/stocks/page.tsx`

  - 현재: 기본 정보(거래소, 시장)와 AI 분석 이력만
  - 목표: 현재가, 등락률, 최소한의 시계열 가격 차트(미니 또는 기본)
  - 파일: `src/app/stocks/[id]/page.tsx`
  - 참고: 백엔드 API에서 가격 데이터 제공 가능 여부 확인 필요

- [ ] **P1-3. 히어로 카피 교체**
  - 현재: "데이터로 스마트하게 투자하세요"
  - 목표: 수치 기반 구체적 메시지. 예) "매일 밤 34개 종목을 AI가 분석합니다" / "데이터가 찾아낸 오늘의 투자 기회"
  - 파일: `src/app/page.tsx`

- [ ] **P1-4. 로그인 페이지에 가입 혜택 안내**
  - 현재: 로그인 폼만
  - 목표: 폼 옆(PC) 또는 상단(모바일)에 "가입하면 얻는 3가지" 표시
  - 파일: `src/app/auth/page.tsx`

- [ ] **P1-5. 전략 카드 N/A 처리**
  - 현재: 구독자 N/A, 평점 N/A 다수
  - 목표: "신규" 뱃지 또는 "- -"으로 대체, N/A 텍스트 제거
  - 파일: `src/components/strategies/StrategyCard.tsx`

- [ ] **P1-6. 홈 페이지 섹션 정리**
  - 현재: 10+ 섹션
  - 목표: 5개 이하로 축소 (히어로 + AI 종목 + 인기 전략 + 통계/소셜프루프 + CTA)
  - 파일: `src/app/page.tsx`

### P2: 1개월 내 (성장 기반 구축)

- [ ] **P2-1. 소셜 프루프 섹션 추가**
  - 목표: 홈 페이지에 얼리 사용자 후기, 분석 적중률, 언론 노출 등
  - 위치: 통계 섹션 인근 또는 CTA 직전

- [ ] **P2-2. 온보딩 플로우 구현**
  - 목표: 가입 직후 → 관심 종목 선택 → 투자 성향 → 맞춤 AI 분석 → 홈
  - 효과: 첫 경험 개선, 개인화된 가치 제공

- [ ] **P2-3. 브랜드 태그라인 확립**
  - 현재: "AI 기반 퀀트 투자 플랫폼" (설명)
  - 목표: 한 문장 브랜드 메시지. 예) "AI가 찾아주는 투자 타이밍"
  - 적용 위치: 메타 태그, OG 이미지, 히어로, 앱 설명

- [ ] **P2-4. BottomNav 라벨 개선**
  - 현재: 홈 / 분석 / 전략 / 종목 / MY
  - 검토: "분석"→"AI추천" 또는 "종목분석" 등 더 직관적 라벨
  - 파일: `src/components/layout/BottomNav.tsx`

- [ ] **P2-5. BETA 뱃지 전략 재수립**
  - 현재: 헤더 + 추천 페이지 배너에 과도 노출
  - 목표: 헤더 뱃지는 유지하되 크기 축소, 페이지 내 배너는 제거 또는 최소화

- [ ] **P2-6. 전략 목록 기본 정렬 최적화**
  - 현재: 마이너스 수익률 전략이 상단에 노출 가능
  - 목표: 기본 정렬을 수익률 높은 순 또는 인기순으로, 마이너스 전략은 하단 배치

## 5. 핵심 원칙 (개선 작업 시 참고)

1. **3초 룰**: 사용자는 3초 안에 "이 서비스가 나에게 가치 있는가"를 판단함
2. **모바일 퍼스트**: 한국 사용자 80%+ 모바일. 모바일 경험이 곧 서비스 경험
3. **부정 > 긍정**: 마이너스 수익률 1개가 플러스 10개보다 인상에 강하게 남음 (손실 회피 편향)
4. **빈 공간 = 미완성**: N/A, 빈 카드, 데이터 없음은 "아직 준비 안 된 서비스" 인상
5. **소셜 프루프**: 초기 스타트업의 최대 약점. 적극적으로 만들어야 함 (내부 데이터라도)
